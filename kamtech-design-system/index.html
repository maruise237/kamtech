Component States Summary

A quick overview of how components behave across different interactive states.

State Visual Cue Examples
Hover Color shifts (darker/lighter background) Buttons, Nav Links
Focus Primary colored ring/shadow outline Inputs, Textareas, Buttons
Loading Spinner animation, reduced opacity, disabled pointer events Buttons (.kt-loading), Cards
Error Red border, red text message, specific focus ring Inputs (.kt-error), Forms, Cards (.kt-error)
Disabled 50% opacity, greyed out background, unclickable Inputs, Buttons
